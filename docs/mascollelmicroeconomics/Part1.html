1 Preference and Choice

1.1 Introduction

The theory of consumer choice forms the foundation of microeconomic analysis. We begin by studying how individuals make decisions when faced with alternatives. The approach is axiomatic: we start with primitive concepts (preferences) and build toward observable behavior (choice and demand).

The central questions we address are:

  • How do we formally represent an individual’s tastes or preferences?
  • Under what conditions can preferences be represented by a utility function?
  • How does a rational consumer choose when constrained by limited resources?

This chapter develops the preference-based approach to consumer theory, which proceeds in three stages: (1) axioms on preference relations, (2) representation by utility functions, and (3) choice under budget constraints.

1.2 Preference Relations

1.2.1 Definition 1 (Preference Relation)

Definition 1.B.1 (Preference Relation)

Let X be the consumption set, representing all possible alternatives available to the consumer. A preference relation \succeq on X is a binary relation where, for any x, y \in X, we write:

  • x \succeq y meaning “x is at least as good as y” (weak preference)

From the weak preference relation \succeq, we derive two additional relations:

  • x \succ y if and only if x \succeq y and not y \succeq x (strict preference: “x is strictly preferred to y”)
  • x \sim y if and only if x \succeq y and y \succeq x (indifference: “x is indifferent to y”)

The consumption set X is typically assumed to be X = \mathbb{R}^L_+, the non-negative orthant of L-dimensional Euclidean space, where L is the number of commodities.

1.2.2 Definition 2 (Rationality Axioms)

Definition 1.B.2 (Rationality Axioms)

A preference relation \succeq on X is rational if it satisfies:

Axiom 1 (Completeness): For all x, y \in X, either x \succeq y or y \succeq x (or both).

Axiom 2 (Transitivity): For all x, y, z \in X, if x \succeq y and y \succeq z, then x \succeq z.

Interpretation: Completeness means the consumer can always compare any two bundles. Transitivity ensures consistency: if you prefer apples to oranges and oranges to bananas, you must prefer apples to bananas.

1.2.3 Proposition 1 (Properties of Rational Preferences)

Proposition 1.B.3 (Properties of Rational Preferences)

If \succeq is a rational preference relation on X, then:

  1. The indifference relation \sim is transitive
  2. The strict preference relation \succ is transitive
  3. The strict preference relation \succ is asymmetric: if x \succ y, then not y \succ x

Proof.

Proof:

  1. Suppose x \sim y and y \sim z. Then x \succeq y, y \succeq x, y \succeq z, and z \succeq y. By transitivity of \succeq, we have x \succeq z and z \succeq x, which means x \sim z.

  2. Suppose x \succ y and y \succ z. Then x \succeq y and not y \succeq x, and y \succeq z and not z \succeq y. By transitivity, x \succeq z. Suppose, for contradiction, that z \succeq x. Then by transitivity with y \succeq z, we get y \succeq x, contradicting x \succ y. Therefore x \succ z.

  3. If x \succ y, then x \succeq y and not y \succeq x. By completeness, since not y \succeq x, we cannot have y \succeq x. Therefore, not y \succ x. ∎

1.3 Continuity and Monotonicity

1.3.1 Definition 3 (Continuity)

Definition 1.C.1 (Continuity)

A preference relation \succeq on X = \mathbb{R}^L_+ is continuous if for any sequence of pairs \{(x^n, y^n)\}_{n=1}^{\infty} with x^n \succeq y^n for all n, and with x^n \to x and y^n \to y, we have x \succeq y.

Equivalently, \succeq is continuous if the sets \{y \in X : y \succeq x\} and \{y \in X : x \succeq y\} are closed for all x \in X.

Interpretation: Continuity means small changes in consumption bundles lead to small changes in preference. There are no “jumps” in preferences.

1.3.2 Definition 4 (Monotonicity)

Definition 1.C.2 (Monotonicity)

A preference relation \succeq on X = \mathbb{R}^L_+ is:

Monotonic if x \geq y (meaning x_\ell \geq y_\ell for all \ell) and x \neq y implies x \succ y.

Strongly monotonic if x \geq y and x \neq y implies x \succ y (more is strictly better).

Locally nonsatiated if for every x \in X and every \varepsilon > 0, there exists y \in X with \|y - x\| < \varepsilon such that y \succ x.

Interpretation: Monotonicity embodies the idea that “more is better.” Local nonsatiation is weaker: in any neighborhood of any bundle, there’s always something better.

1.3.3 Proposition 2 (Monotonicity Relationships)

Proposition 1.C.3 (Monotonicity Relationships)

  1. Strong monotonicity implies monotonicity
  2. Strong monotonicity implies local nonsatiation
  3. Monotonicity does not imply local nonsatiation

Proof.

Proof sketch:

  1. and (ii) are immediate from definitions. For (iii), consider preferences where the consumer is satiated at some point x^* and x \succeq y if and only if \|x - x^*\| \leq \|y - y^*\|. This is monotonic but not locally nonsatiated at x^*. ∎

1.4 Convexity

1.4.1 Definition 5 (Convex Preferences)

Definition 1.D.1 (Convex Preferences)

A preference relation \succeq on X is convex if for all x, y, z \in X with x \succeq z and y \succeq z, and for all \alpha \in [0,1]:

\alpha x + (1-\alpha)y \succeq z

The preference relation is strictly convex if for all x, y, z \in X with x \succeq z, y \succeq z, x \neq y, and for all \alpha \in (0,1):

\alpha x + (1-\alpha)y \succ z

Interpretation: Convexity means “averages are preferred to extremes” or equivalently, consumers prefer diversity. If you like both bundles x and y at least as much as z, you also like mixtures of x and y at least as much as z.

1.4.2 Proposition 3 (Convexity and Indifference Sets)

Proposition 1.D.2 (Convexity and Indifference Sets)

A preference relation \succeq is convex if and only if for every x \in X, the upper contour set \{y \in X : y \succeq x\} is convex.

Proof.

Proof:

(\Rightarrow) Suppose \succeq is convex and let y, z \in \{w : w \succeq x\}. Then y \succeq x and z \succeq x. By convexity, for any \alpha \in [0,1], \alpha y + (1-\alpha)z \succeq x. Thus \alpha y + (1-\alpha)z is in the upper contour set, proving it’s convex.

(\Leftarrow) Suppose all upper contour sets are convex. Take x \succeq z and y \succeq z. Then x, y \in \{w : w \succeq z\}. Since this set is convex, \alpha x + (1-\alpha)y \in \{w : w \succeq z\} for all \alpha \in [0,1], meaning \alpha x + (1-\alpha)y \succeq z. ∎

2 Utility Representation

2.1 Utility Functions

2.1.1 Definition 6 (Utility Function)

Definition 2.A.1 (Utility Function)

A function u: X \to \mathbb{R} is a utility function representing preference relation \succeq if for all x, y \in X:

x \succeq y \Longleftrightarrow u(x) \geq u(y)

Interpretation: A utility function assigns numbers to bundles such that the ordering of numbers matches the preference ordering. The numbers themselves are ordinal (only rankings matter), not cardinal.

2.1.2 Proposition 4 (Utility and Derived Relations)

Proposition 2.A.2 (Utility and Derived Relations)

If u represents \succeq, then:

  • x \succ y if and only if u(x) > u(y)
  • x \sim y if and only if u(x) = u(y)

Proof.

Proof:

Direct from the definition of utility representation and the definitions of \succ and \sim. ∎

2.1.3 Proposition 5 (Positive Monotonic Transformations)

Proposition 2.A.3 (Positive Monotonic Transformations)

If u represents \succeq and f: \mathbb{R} \to \mathbb{R} is strictly increasing, then v = f \circ u also represents \succeq.

Proof.

Proof:

For any x, y \in X:

x \succeq y \Longleftrightarrow u(x) \geq u(y) \Longleftrightarrow f(u(x)) \geq f(u(y)) \Longleftrightarrow v(x) \geq v(y)

The second equivalence holds because f is strictly increasing. ∎

Implication: Utility is ordinal. If u works, so does u^2, \sqrt{u}, \ln u, etc. (when appropriate). Only the ordering matters.

2.2 The Representation Theorem

2.2.1 Theorem 1 (Existence of Utility Representation)

Theorem 2.B.1 (Existence of Utility Representation)

If X = \mathbb{R}^L_+ and \succeq is rational and continuous, then there exists a continuous utility function u: X \to \mathbb{R} representing \succeq.

Proof.

Sketch of Proof:

The proof constructs u explicitly. For X = \mathbb{R}^L_+, define:

u(x) = \min\{\alpha \in \mathbb{R} : \alpha e \succeq x\}

where e = (1,1,\ldots,1). Continuity of \succeq ensures this minimum exists and that u is continuous. Rationality ensures u represents \succeq. ∎

Importance: This theorem justifies the common practice of working with utility functions rather than preference relations. Under weak assumptions, the two approaches are equivalent.

2.2.2 Proposition 6 (Uniqueness Up to Monotonic Transformation)

Proposition 2.B.2 (Uniqueness Up to Monotonic Transformation)

If both u and v represent \succeq, then there exists a strictly increasing function f such that v = f \circ u.

Proof.

Proof:

Define f: u(X) \to \mathbb{R} by f(u(x)) = v(x). This is well-defined because if u(x) = u(y), then x \sim y, so v(x) = v(y). The function f is strictly increasing: if u(x) > u(y), then x \succ y, so v(x) > v(y). ∎

2.3 Properties of Utility Functions

2.3.1 Definition 7 (Quasi-concavity)

Definition 2.C.1 (Quasi-concavity)

A function u: X \to \mathbb{R} is quasi-concave if for all x, y \in X and \alpha \in [0,1]:

u(\alpha x + (1-\alpha)y) \geq \min\{u(x), u(y)\}

It is strictly quasi-concave if for x \neq y and \alpha \in (0,1):

u(\alpha x + (1-\alpha)y) > \min\{u(x), u(y)\}

2.3.2 Proposition 7 (Quasi-concavity and Convex Preferences)

Proposition 2.C.2 (Quasi-concavity and Convex Preferences)

A preference relation \succeq is convex if and only if any utility function representing it is quasi-concave. It is strictly convex if and only if any utility function representing it is strictly quasi-concave.

Proof.

Proof (convexity \Leftrightarrow quasi-concavity):

(\Rightarrow) Suppose \succeq is convex and u represents it. Let u(x) \geq u(z) and u(y) \geq u(z). Then x \succeq z and y \succeq z. By convexity, \alpha x + (1-\alpha)y \succeq z, so u(\alpha x + (1-\alpha)y) \geq u(z) = \min\{u(x), u(y), u(z)\} \geq \min\{u(x), u(y)\}.

(\Leftarrow) Suppose u is quasi-concave. If x \succeq z and y \succeq z, then u(x) \geq u(z) and u(y) \geq u(z). By quasi-concavity, u(\alpha x + (1-\alpha)y) \geq \min\{u(x), u(y)\} \geq u(z), so \alpha x + (1-\alpha)y \succeq z. ∎

3 Exercises

3.1 Exercise Set 1.B

3.1.1 Exercise 1.B.1 (Verifying Rationality)

Exercise 1.B.1 (Verifying Rationality)

Consider the consumption set X = \mathbb{R}^2_+. For each preference relation below, determine whether it is rational (complete and transitive):

  1. x \succeq y if and only if x_1 + x_2 \geq y_1 + y_2

  2. x \succeq y if and only if x_1 \geq y_1 or x_2 \geq y_2

  3. x \succeq y if and only if \min\{x_1, x_2\} \geq \min\{y_1, y_2\}

Solutions:

  1. Rational. Completeness: For any x, y, either x_1 + x_2 \geq y_1 + y_2 or y_1 + y_2 \geq x_1 + x_2. Transitivity: If x_1 + x_2 \geq y_1 + y_2 and y_1 + y_2 \geq z_1 + z_2, then x_1 + x_2 \geq z_1 + z_2.

  2. Rational. One can verify both completeness and transitivity hold using the definition of “or” in logic.

  3. Rational. Similar argument to (a), treating \min\{x_1, x_2\} as a single aggregate.

3.1.2 Exercise 1.B.2 (Lexicographic Preferences)

Exercise 1.B.2 (Lexicographic Preferences)

Consider lexicographic preferences on \mathbb{R}^2_+: x \succeq y if and only if [x_1 > y_1] or [x_1 = y_1 and x_2 \geq y_2].

  1. Show that \succeq is rational.
  2. Show that \succeq cannot be represented by a utility function.

Solutions:

  1. Completeness: For any x, y, either x_1 > y_1, or y_1 > x_1, or x_1 = y_1. In the last case, either x_2 \geq y_2 or y_2 \geq x_2. Thus either x \succeq y or y \succeq x.

Transitivity: Suppose x \succeq y and y \succeq z.

Case 1: x_1 > y_1. Then x_1 > y_1 \geq z_1 (where if y_1 > z_1 we’re done, and if y_1 = z_1 then y_1 \geq z_1), so x \succeq z.

Case 2: x_1 = y_1 and x_2 \geq y_2. If y_1 > z_1, then x_1 = y_1 > z_1, so x \succeq z. If y_1 = z_1, then x_1 = y_1 = z_1, and from y \succeq z we have y_2 \geq z_2, so x_2 \geq y_2 \geq z_2, giving x \succeq z.

  1. Suppose u represents \succeq. For each x_1 \in [0,1], consider the interval I(x_1) = u(\{x_1\} \times \mathbb{R}_+). Since (x_1, a) \succ (x_1, b) whenever a > b, u must map each vertical line to an interval of real numbers. For x_1 > y_1, we need (x_1, 0) \succ (y_1, a) for all a, so \inf I(x_1) > \sup I(y_1). Thus the intervals must be disjoint. But we have uncountably many disjoint intervals in \mathbb{R}, which is impossible since \mathbb{R} contains only countably many disjoint intervals (each must contain a distinct rational). Contradiction.

3.2 Exercise Set 2.A

3.2.1 Exercise 2.A.1 (Computing Utility Functions)

Exercise 2.A.1 (Computing Utility Functions)

For each preference relation, find a utility function that represents it:

  1. x \succeq y if x_1x_2 \geq y_1y_2 (Cobb-Douglas)

  2. x \succeq y if \min\{x_1, x_2\} \geq \min\{y_1, y_2\} (Leontief)

  3. x \succeq y if ax_1 + bx_2 \geq ay_1 + by_2 where a, b > 0 (Linear)

Solutions:

  1. u(x) = x_1x_2

  2. u(x) = \min\{x_1, x_2\}

  3. u(x) = ax_1 + bx_2

3.2.2 Exercise 2.A.2 (Monotonic Transformations)

Exercise 2.A.2 (Monotonic Transformations)

Let u(x_1, x_2) = x_1x_2 represent some preferences \succeq.

  1. Show that v(x_1, x_2) = \ln(x_1) + \ln(x_2) represents the same preferences.

  2. Do u and v have the same indifference curves? Explain.

  3. Find another utility function representing \succeq.

Solutions:

  1. Note that v(x) = \ln(u(x)). Since \ln is strictly increasing on \mathbb{R}_{++}, by Proposition 2.A.3, v represents \succeq.

  2. Yes. Indifference curves are level sets of the utility function. For any \bar{u}, the set \{x : u(x) = \bar{u}\} equals \{x : \ln(u(x)) = \ln(\bar{u})\} = \{x : v(x) = \ln(\bar{u})\}, so they define the same sets.

  3. w(x_1, x_2) = (x_1x_2)^2 or w(x_1, x_2) = \sqrt{x_1x_2} or infinitely many others.

These notes provide a rigorous foundation for consumer theory. The axiomatic approach ensures logical consistency while the representation theorems connect abstract preferences to concrete utility functions used in applications.
